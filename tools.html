操作工具／機具

使用工具時請小心，不要傷害自己。
Be careful when using tools and do not hurt yourself.

未取得老師的同意之前，不可操作機器。
Do not operate the machine without the teacher’s permission.

不可以戴手套操作機器。
Do not operate the machine while wearing gloves.

操作機器時，確保旁邊沒有人站得太近。
Ensure that no one stands too close to the machine while operating it.

不要太靠近運轉中的機器。
Do not get too close to the machine while it is running.

不使用機器時，應關閉電源。
Turn off the power when the machine is not in use.

請依序使用機器。
Please use the machine in order.

工具用完後，請擺放整齊。
Please put the tools away neatly after using them.

先鑽孔，再鋸切。
Drill first, then saw.

若有任何問題請報告老師。
Please report to the teacher if you have any problems.

老師，我的[材料]壞掉了。
Teacher, my [material] is broken.
